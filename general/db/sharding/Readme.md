# Демо проект по шардированию

## Запуск инфры

```bash
docker compose up -d
```

## Запуск приложения

```bash
go run ./cmd/app
```

## Что осталось

- Выбор ключа шардирования и Визуализация распределения ключей по шардам (чтобы показать проблемы)
- Шардирование с промежуточной таблицей
- Виртуальные узлы
- Ввод нового узла
